# OwnersHQ — Restaurant Platform

The all-in-one restaurant management platform.

## Modules
- **Command Center** — Business health scores, action items, activity feed
- **MOE Inventory** — AI-powered inventory tracking (connected via Supabase)
- **Menu Builder** — AI menu creation, pricing, descriptions, placement strategy
- **Recipe Cards** — Kitchen reference with printable PDFs
- **Prep & Menu Costing** — Master product list, food cost analysis, margins
- **P&L Reports** — Daily, weekly, monthly profit & loss reporting
- **Staff & Prep Log** — Coming soon

## Tech Stack
- React 18 + Vite
- Supabase (shared with MOE)
- Vercel deployment
- Claude API for AI features

## Setup
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → Vercel auto-deploys.
